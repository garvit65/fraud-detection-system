# Financial Fraud Detection System

## Overview
This project is a real-time financial fraud detection system using machine learning. It analyzes transaction data and detects fraudulent activities.

## Features
- Real-time transaction simulation
- Machine learning-based fraud detection
- Risk classification (Low, Medium, High)
- Interactive dashboard using Dash
- Data visualization (charts and graphs)

## Technologies Used
- Python
- Scikit-learn
- Plotly Dash
- Pandas

## Dataset
- PaySim Dataset (synthetic financial data)

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py
